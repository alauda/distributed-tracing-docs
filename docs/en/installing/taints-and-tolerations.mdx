# Using Taints and Tolerations

You can use taints and tolerations to control which nodes the OpenTelemetry Collector and Jaeger pods are scheduled on. This is useful when you want to dedicate specific infrastructure nodes for distributed tracing workloads.

For detailed instructions on configuring `nodeSelector` and `tolerations` in the `OpenTelemetryCollector` custom resource, see <ExternalSiteLink name="opentelemetry" href="/installing/taints-and-tolerations.html" children="Using Taints and Tolerations" /> in the Alauda Build of OpenTelemetry v2 documentation.
