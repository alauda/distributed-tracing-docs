# Alauda Build of Jaeger Lifecycle Policy

## Version Lifecycle Timeline

Below is the life cycle schedule for released versions of `Alauda Build of Jaeger`:

| Version  | Release Date  | End of Support  |
| -------- | ------------- | --------------- |
| v2.0.z   | 2026-04-21    | 2027-10-21      |

## Release Policy

A new version of `Alauda Build of Jaeger` is released every 4 months.

## Maintenance Policy

Alauda provides 18 months of maintenance support for each released version of `Alauda Build of Jaeger`.

During the maintenance period, Alauda will provide the following services to customers:

1. Track the patch versions and promptly provide patches containing bug fixes and security updates.
2. Provide security updates that comply with Alauda's security standards.
3. Assist customers in smoothly upgrading to new versions.
